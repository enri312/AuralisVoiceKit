Instalacion

El paquete base no instala dependencias nativas. Esa decision mantiene el core compatible con Python moderno y con versiones futuras razonables.

python -m pip install auralisvoicekit

Desde un repositorio clonado, el quickstart de PyPI valida el flujo base sin extras.

python examples/pypi_quickstart.py --json

Extras opcionales:

python -m pip install "auralisvoicekit[sounddevice]"
python -m pip install "auralisvoicekit[pyaudio]"
python -m pip install "auralisvoicekit[openai]"
python -m pip install "auralisvoicekit[whisper]"

Durante desarrollo local:

python -m pip install -e .
python -m auralis_voicekit.cli doctor
python -m unittest discover -s tests

Inicio rapido

El backend por defecto es null. Permite verificar integraciones sin microfono, red, modelos ni credenciales.

from auralis_voicekit import AuralisVoiceKit, AudioChunk, AudioFormat

kit = AuralisVoiceKit()
chunk = AudioChunk(
    data=b"\x00\x00" * 16000,
    format=AudioFormat(sample_rate=16000, channels=1, sample_width=2),
)

result = kit.transcribe(chunk)
print(result.text)

Para probar un flujo mas cercano a una integracion real, usa examples/pypi_quickstart.py. Genera un WAV sintetico, crea una sesion de voz, segmenta y transcribe con null.

python examples/pypi_quickstart.py --json

Modelos

Simbolo Uso Campos principales
AudioEncoding Enum de codificacion de audio. PCM16, FLOAT32
AudioFormat Formato de un chunk de audio. sample_rate, channels, sample_width, encoding, bytes_per_second
AudioChunk Bloque de audio con metadata. data, format, timestamp, metadata, duration_seconds
TranscriptResult Resultado de transcripcion. text, confidence, language, is_final, source, metadata
AudioDevice Dispositivo de entrada reportado por un backend. id, name, kind, channels, host_api, is_default, metadata

El timeout de transcripcion es opcional. transcription_timeout_seconds tambien se puede configurar con AURALIS_TRANSCRIPTION_TIMEOUT_SECONDS o --timeout-seconds en la CLI; el backend openai lo pasa al cliente oficial como limite de red. English: local Whisper runs remain local/model-bound and are not interrupted by this setting.

AuralisVoiceKit

AuralisVoiceKit es la fachada principal. Compone configuracion, registro de backends y bus de eventos.

Metodo Descripcion
start_capture(on_audio=None) Abre el backend de captura y emite eventos capture.started y audio.chunk.
stop_capture() Detiene captura y emite capture.stopped.
transcribe(chunk) Transcribe un AudioChunk con el backend seleccionado.
speak(text) Envia texto al backend de salida. Con output_backend="system" puede usar voz, velocidad y volumen.
queue_speech(text) / queue_speech_many(texts) Encola una o varias frases para salida secuencial sin exponer texto en eventos.
drain_output_queue(limit=None) Drena la cola en orden usando el backend actual; si el backend falla, conserva el item actual para reintento.
clear_output_queue() / output_queue_size Limpia pendientes o consulta cuantos textos siguen esperando salida.
backend_report() Devuelve informacion de backends registrados.

Sesiones

VoiceSession agrupa lectura o captura, segmentacion y transcripcion por turnos.

from auralis_voicekit import AuralisVoiceKit, VoiceKitConfig, VoiceSession, VoiceSessionConfig

kit = AuralisVoiceKit(
    VoiceKitConfig(transcription_backend="whisper", transcription_model="base")
)

with VoiceSession(kit, VoiceSessionConfig(chunk_duration_ms=50)) as session:
    turns = session.transcribe_file("sample.mp3")
    for turn in turns:
        print(turn.index, turn.text)
Simbolo Uso
VoiceSessionConfig Controla chunking, VAD, normalizacion, limite de turnos, ejecutable ffmpeg, activation_phrases y sensibilidad de mayusculas para activacion.
VoiceTurn Une un VoiceSegment y un TranscriptResult. Expone text, duration_seconds, rms y to_dict().
VoiceSession Expone segment_chunks, transcribe_segments, transcribe_chunks, transcribe_wav, transcribe_file, turn_is_activated, capture_for, listen_once, cancel y close.

Las llamadas de transcripcion de sesion aceptan require_activation=True para devolver solo turnos activados. La activacion puede venir de activation_phrases o de un activation_hook externo. English: activation hooks are opt-in and keep the default behavior unchanged.

session = VoiceSession(
    kit,
    VoiceSessionConfig(activation_phrases=("auralis",)),
)
turns = session.transcribe_file("sample.mp3", require_activation=True)

external_turns = session.transcribe_chunks(
    chunks,
    require_activation=True,
    activation_hook=lambda turn: turn.text.startswith("ok asistente"),
)

Eventos y privacidad

EventBus permite observar el flujo sin acoplarse a un backend. Con privacy_mode=True, los eventos evitan audio crudo y texto transcrito. Para logs persistentes, PrivacyEventLogger escribe JSONL con payload sanitizado.

from auralis_voicekit import (
    EventBus,
    PrivacyEventLogger,
    PrivacyLogConfig,
    VoiceEvent,
    VoiceEventType,
    event_to_log_record,
    sanitize_event_payload,
)

events = EventBus()
unsubscribe = events.subscribe(
    VoiceEventType.ANY,
    lambda event: print(event.type, event.payload),
)
event = events.emit(VoiceEventType.ERROR, {"message": "demo"})
record = event_to_log_record(event)
safe_payload = sanitize_event_payload({"text": "privado", "backend": "null"})
unsubscribe()
with PrivacyEventLogger("auralis-events.jsonl") as logger:
    unsubscribe = logger.subscribe(events)
    events.emit(VoiceEventType.TRANSCRIPTION_COMPLETED, {"text": "privado"})
    unsubscribe()

debug_config = PrivacyLogConfig(privacy_mode=False)

Salida personalizada

Un backend de salida custom implementa SpeechOutputBackend, reporta BackendInfo(kind="output") y se registra con BackendRegistry.register_output(). La guia completa vive en CUSTOM_OUTPUT_BACKENDS.md.

from auralis_voicekit import AuralisVoiceKit, VoiceKitConfig
from auralis_voicekit.backends import BackendInfo, create_default_registry

class QueueOutputBackend:
    name = "queue"

    def __init__(self):
        self.items = []

    def info(self):
        return BackendInfo(name=self.name, kind="output")

    def speak(self, text, config):
        if text.strip():
            self.items.append(text)

backend = QueueOutputBackend()
registry = create_default_registry()
registry.register_output("queue", lambda: backend)
kit = AuralisVoiceKit(VoiceKitConfig(output_backend="queue"), registry=registry)
kit.speak("Hola desde una salida custom")

Registro custom

from auralis_voicekit import AuralisVoiceKit, TranscriptResult, VoiceKitConfig
from auralis_voicekit.backends import BackendRegistry, create_default_registry

class MyTranscriber:
    name = "my-transcriber"

    def info(self):
        from auralis_voicekit.backends.base import BackendInfo
        return BackendInfo(name=self.name, kind="transcription")

    def transcribe(self, chunk, config):
        return TranscriptResult(text="hola", language=config.language, source=self.name)

registry = create_default_registry()
registry.register_transcription("my-transcriber", MyTranscriber)
kit = AuralisVoiceKit(
    VoiceKitConfig(transcription_backend="my-transcriber"),
    registry=registry,
)
